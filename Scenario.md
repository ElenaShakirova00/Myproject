# Сценарий использования для проекта пиццерии ## Название сценария: Заказ пиццы ### Участники: - Клиент - Оператор ### Предусловия: 1. Клиент должен быть зарегистрирован в системе.2. Оператор должен быть онлайн. ### Основной поток: 1. Клиент заходит на сайт. 2. Выбирает пиццу из меню. 3. Добавляет пиццу в корзину. 4. Клиент переходит к оформлению заказа.
5. **Клиент получает уведомление о новом заказе.** 6. Оператор получает уведомление о новом заказе. ### Альтернативный поток: - Если пицца закончилась, клиент получает уведомление о том, что пиццы нет в наличии.
