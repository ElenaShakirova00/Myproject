# Сценарий использования для проекта пиццерии ## Название сценария: Заказ пиццы ### Участники: - Клиент - Система### Предусловия: 1. Клиент должен быть зарегистрирован в системе.2. Система должна работать ### Основной поток: 1. Клиент заходит на сайт. 2. Выбирает пиццу из меню. 3. Добавляет пиццу в корзину. 4. Клиент переходит к оформлению заказа.
5. Клиент производит оплату 
6. **Клиент получает уведомление о подтверждении заказа.**7. Система получает уведомление о новом заказе. ### Альтернативный поток: - Если пицца закончилась, клиент получает уведомление о том, что пиццы нет в наличии.
